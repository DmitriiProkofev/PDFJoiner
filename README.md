# PDFJoiner
Пример PDFJoiner:

1. Добавление изображения "галочки" в левый верхний угол исходного pdf;
2. Объединение pdf файлов в один, с возможностью навигации по закладкам;
3. Настройка порядка объединения файлов.

Запуск:

 - Не требует дополнительных настроек.

Структура приложения:

 - PDFJoiner: основная программа.
